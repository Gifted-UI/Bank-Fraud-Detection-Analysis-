# Bank Fraud Detection Analysis

## Project Overview

This project presents a comprehensive fraud detection analysis built using Power BI. The dashboard helps financial institutions monitor fraudulent transactions, identify high-risk customers, analyze transaction patterns, and support data-driven decision-making.

---

## Business Problem

Financial institutions process millions of transactions daily, making it difficult to detect fraudulent activities manually. This project analyzes banking transaction data to uncover fraud patterns, monitor key performance indicators, and improve fraud prevention strategies.

---

## Project Objectives

- Detect fraudulent transactions
- Monitor fraud trends over time
- Compare fraud across different countries
- Identify high-risk customers
- Analyze transaction amounts and account balances
- Build an interactive dashboard for decision-making

## Key Insights

- Fraud cases are concentrated within specific transaction types.
- Some countries recorded significantly higher fraud rates than others.
- High-value transactions are more likely to be fraudulent.
- A small percentage of customers account for a large proportion of fraud cases.
- Monitoring transaction patterns can significantly improve fraud detection.
